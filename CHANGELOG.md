# Changelog

## v1.0 — Initial AI Red-Teaming Lab

### Added
- Threat model defining adversaries, assets, and target systems
- Failure taxonomy for classifying AI model weaknesses
- Attack techniques framework (prompt injection, indirect injection, agent misuse)
- Risk severity classification model
- Adversarial attack scenarios:
  - Role confusion prompt injection
  - RAG poisoned document (indirect injection)
  - Agent tool misuse
- Evaluation framework with standardized outcome signals
- Adversarial prompt dataset for testing
- Attack coverage matrix summarizing test scenarios
- Structured finding template for vulnerability reporting
- Sample findings demonstrating red-team results

- README with project overview and methodology
